#### RUN ON VPS/UBUNTU/WINDOWS

1. Install NodeJs,ffmpeg
2. Installation
   ```
   git clone https://github.com/AstroX10/Xstro.git
   cd Xstro
   yarn
   ```
3. Configuration
   ```
   echo "
     SESSION_ID = null
     SUDO = null
     PREFIX = .
     MODE = private
     LOG_MSG = true" > config.env
   ```

- Start
  ```
  npm start
  ```
- Stop
  ```
  npm stop
  ```
